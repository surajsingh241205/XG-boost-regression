# XGBoost Regression

This project demonstrates the implementation of **XGBoost Regression**
using Python, scikit-learn, and XGBoost on a synthetic dataset.

XGBoost stands for **Extreme Gradient Boosting** and is one of the most powerful
machine learning algorithms used in real-world applications, Kaggle competitions,
and production systems.

It builds multiple decision trees sequentially, where each new tree tries to
correct the errors of the previous tree, resulting in very high accuracy.

## 📘 What is XGBoost?

XGBoost is an advanced version of Gradient Boosting.

It builds models step by step, where each new model
tries to reduce the error made by the previous model.

Compared to normal Gradient Boosting, XGBoost is:

- Faster
- More accurate
- Optimized for performance
- Widely used in real-world ML projects

## 🚀 Why XGBoost is Popular

- High accuracy
- Handles large datasets
- Works well with structured data
- Used in Kaggle winning solutions
